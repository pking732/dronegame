# FPV: Spectrum (Antigravity/Gravity Edition)

Краткое описание игры "FPV: Spectrum": Симулятор боевых дронов от лица оператора. Упор на реалистичную физику поведения дрона (FPV) и аналоговый визуал (помехи, эффекты с камеры).

## Файловая структура (Логическое разделение)

Структура проекта строго разделяет физику, визуализацию и логику:

* `/src/core` — Физический движок (Rapier.js) и контроллеры полета.
* `/src/entities` — Модели дронов, боевой техники (танки, БТР) и пехоты.
* `/src/ui` — Интерфейс оператора (HUD), главное меню, "выкрашенный" или "украшенный" журнал.
* `/src/effects` — Постобработка и эффекты (шейдеры аналогового шума, взрывы, пыль).
* `/src/store` — Глобальное состояние игры (Zustand) и интеграция с базой данных (Supabase).
